## Data Persistence
The web service requires the use of an SQL database to store character and word use.  
Schema:  
* WORD_USE  
  * WORD : VARCHAR(50)  
  * USES : INT(10)  
* CHAR_USE  
  * LETTER : VARCHAR(50)  
  * USES : INT(10)
